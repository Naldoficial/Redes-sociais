# 📱 Ferramentas para Redes Sociais — OSINT
Repositório pessoal com ferramentas cadastradas para **buscar apenas informações PÚBLICAS** de perfis em redes sociais.

## 📌 O que é esse projeto?
Essa é a minha coleção de ferramentas legais, que mostram só o que as pessoas já escolheram deixar visível para todo mundo na internet. Não serve para acessar contas privadas, roubar dados ou espionar — tudo feito com respeito às regras e à privacidade.

## 📂 O que tem aqui?
- Arquivos de configuração de ferramentas de busca de dados públicos
- Guias rápidos de uso seguro
- Atualizações conforme novas ferramentas são adicionadas

## ⚠️ Regras importantes
✅ **Só uso legal**: Apenas para fins de estudo, segurança e consulta de dados que já estão públicos.
❌ **Proibido**: Usar para invadir contas, perseguir pessoas ou usar informações para prejudicar outros — isso é crime no Brasil.
